# 영상 편집 사용성 개선 디자인

## 목표

영상 편집 화면의 속도/텍스트/스티커 기능을 사용성 있게 개선한다. 구간별 속도 조절, 텍스트 멀티트랙 타임라인, 이모지 스티커 + 시간/제스처 지원을 추가한다.

## 현재 상태

- **속도:** 전체 영상에 0.5x/1x/2x/4x 균일 적용. `SpeedSegmentsNotifier`에 구간 관리 인프라 존재하나 UI 미노출.
- **텍스트:** 여러 개 추가 가능. 단일 트랙 타임라인(`SubtitleTimelineTrack`)에 블록 표시, 양쪽 핸들로 시간 조절. 리치 스타일링 지원.
- **스티커:** V-Grade 스티커만. 드래그 이동 + 길게 눌러 삭제. 크기/회전/시간 조절 없음.
- **레이아웃:** 하단 아이콘 버튼(속도/스티커/텍스트) → 바텀시트. 트림슬라이더 + 배속바 + 자막트랙이 항상 표시.

---

## 1. 편집 화면 레이아웃 — 탭 바 방식

하단 도구 영역을 탭 바로 전환한다. 각 탭 선택 시 타임라인 영역이 해당 도구 UI로 전환된다.

```
┌──────────────────────────────┐
│        상단 바 (편집/내보내기)    │
│──────────────────────────────│
│                              │
│      영상 프리뷰 + 오버레이      │
│                              │
│──────────────────────────────│
│   [도구별 타임라인/조작 영역]     │  ← 선택한 탭에 따라 전환
│──────────────────────────────│
│  트림  │  속도  │ 텍스트 │ 스티커 │  ← 탭 바
└──────────────────────────────┘
```

- 트림 탭: 기존 `TrimSlider` + `TrimTimeline`
- 속도 탭: 구간별 속도 타임라인
- 텍스트 탭: 멀티트랙 타임라인 + 추가 버튼
- 스티커 탭: 멀티트랙 타임라인 + 추가 버튼

새 위젯: `editor_tab_bar.dart`
새 Provider: `selectedEditorTabProvider`

## 2. 속도 — 구간별 속도 조절

속도 탭 선택 시 인터랙티브 타임라인 표시.

```
┌─────────────────────────────────────┐
│  [0.5x] [1x] [2x] [4x]   [✂ 분할]  │
│─────────────────────────────────────│
│  ┌──────┬──────────┬────────┐       │
│  │ 1x   │   0.5x   │  2x   │       │  ← 구간별 색상 구분
│  └──────┴──────────┴────────┘       │
│         ▲ 재생 위치                   │
└─────────────────────────────────────┘
```

### 조작 방식
- **분할:** "✂ 분할" 버튼으로 현재 재생 위치 기준 좌/우 분할
- **속도 적용:** 구간 탭 선택 → 상단 속도 버튼(0.5x/1x/2x/4x)으로 지정
- **구간 병합:** 인접한 동일 속도 구간 자동 병합
- **경계 조절:** 구간 경계선 좌우 드래그로 미세 조절
- **색상:** 0.5x=파랑, 1x=초록, 2x=주황, 4x=빨강

### 구현
- 기존 `SpeedSegmentsNotifier`에 `splitAt(Duration)`, `mergeAdjacent()` 메서드 추가
- `SpeedSegmentBar`를 인터랙티브 위젯으로 확장 → `SpeedSegmentTimeline`
- FFmpeg 처리: 이미 다중 구간 지원 (`ffmpeg_command_builder.dart`)

## 3. 텍스트 — 멀티트랙 타임라인

텍스트 탭 선택 시 텍스트별 트랙 표시.

```
┌─────────────────────────────────────┐
│  [+ 텍스트 추가]                      │
│─────────────────────────────────────│
│  T1 ┌──"완등!"──────┐               │
│  T2      ┌──"V3 성공"──────┐        │
│  T3               ┌──"핵심구간"─┐    │
│─────────────────────────────────────│
│         ▲ 재생 위치                   │
└─────────────────────────────────────┘
```

### 조작 방식
- **트랙 배치:** 텍스트 하나당 트랙 한 줄, 5개 이상 시 스크롤
- **구간 조절:** 블록 좌/우 핸들 드래그로 시작/끝 시간 조절
- **블록 이동:** 블록 본체 좌우 드래그 → 시작/끝 동시 이동 (구간 길이 유지)
- **편집:** 블록 탭 → `SubtitleEditorSheet` 열림
- **삭제:** 블록 길게 누르면 삭제 확인
- **추가:** "+" 버튼 → `SubtitleEditorSheet`, 기본 시간 = 현재 재생 위치 ~ +3초

### 구현
- 기존 `SubtitleTimelineTrack`을 `TextMultiTrackTimeline`으로 확장
- 블록 본체 드래그(구간 이동) 로직 추가
- 기존 `SubtitlesNotifier`, `SubtitleEditorSheet` 재활용

## 4. 스티커 — 이모지 + 시간 + 제스처

### 4-1. 이모지/프리셋 스티커

`OverlayStickerSheet`를 탭 구조로 확장:

```
┌─────────────────────────────────────┐
│  [등급]  [이모지]                     │
│─────────────────────────────────────│
│  등급 탭: 기존 V-Grade UI             │
│  이모지 탭: 카테고리별 그리드           │
│    클라이밍: 🧗💪🔥⛰️🏔️🪨🎯👏      │
│    감정:   😆😤🥲😎🤯🫣             │
│    기타:   ⭐❤️✨🎉👍              │
└─────────────────────────────────────┘
```

- 이모지는 텍스트 기반 `OverlayItem`으로 처리 (별도 에셋 불필요)
- fontSize 48~64로 렌더링

### 4-2. 스티커 표시 시간 조절

`OverlayItem`에 `startTime`/`endTime` 필드 추가. 스티커 탭에 멀티트랙 타임라인 표시:

```
┌─────────────────────────────────────┐
│  [+ 스티커 추가]                      │
│─────────────────────────────────────│
│  S1 ┌──"V3"──────────┐              │
│  S2      ┌──"🔥"──────────┐         │
│─────────────────────────────────────│
│         ▲ 재생 위치                   │
└─────────────────────────────────────┘
```

- 텍스트 트랙과 동일한 UX (핸들 드래그, 블록 이동)
- FFmpeg: `enable='between(t,...)'` 필터 적용 (텍스트와 동일 방식)

### 4-3. 크기 조절 + 회전 제스처

`OverlayLayer`에 제스처 추가:
- **핀치 줌:** 두 손가락으로 크기 조절 → `fontSize` 업데이트
- **두 손가락 회전:** 회전 각도 → `rotation` 필드 업데이트
- 선택된 스티커에 리사이즈/회전 핸들 표시 (코너에 원형 핸들)

### 모델 변경

```dart
class OverlayItem {
  // 기존 필드 유지
  final Duration? startTime;   // 추가
  final Duration? endTime;     // 추가
  final double rotation;       // 추가 (라디안, 기본값 0.0)
}
```
