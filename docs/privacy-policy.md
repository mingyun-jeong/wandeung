# 개인정보처리방침

**클림픽** (이하 "앱" 또는 "서비스")은 이용자의 개인정보를 소중히 보호하며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 개인정보처리방침은 이용자가 제공하는 개인정보가 어떻게 수집, 이용, 보관, 파기되는지를 안내합니다.

**시행일:** 2026년 3월 8일

## 6. 개인정보의 파기 절차 및 방법

### 6-1. 파기 절차

이용자의 개인정보는 수집 목적이 달성되거나 회원 탈퇴 요청 시 지체 없이 파기합니다.

1. 이용자가 회원 탈퇴를 요청합니다.
2. 데이터베이스에 저장된 모든 클라이밍 기록이 삭제됩니다.
3. Supabase 인증 시스템에서 계정 정보가 삭제됩니다.
4. 기기에 저장된 영상 파일은 서버에 존재하지 않으므로, 이용자가 직접 기기에서 삭제하거나 앱을 삭제하여 제거할 수 있습니다.

### 6-2. 파기 방법

| 정보 유형 | 파기 방법 |
|-----------|-----------|
| 데이터베이스 기록 | Supabase 데이터베이스에서 해당 이용자의 모든 레코드를 영구 삭제 (DELETE) |
| 인증 정보 | Supabase Auth에서 해당 이용자의 계정을 영구 삭제 |
| 영상 파일 | 기기에만 저장되어 있으므로 이용자가 직접 삭제 또는 앱 삭제 시 제거 |

## 8. 카메라 및 마이크 접근 권한

### 8-1. 접근 권한의 목적

| 접근 권한 | 목적 | 필수 여부 |
|-----------|------|-----------|
| 카메라 | 클라이밍 영상 촬영 | 선택 |
| 마이크 | 클라이밍 영상 촬영 시 음성 녹음 | 선택 |

### 8-2. 접근 권한 관련 안내

- 카메라와 마이크 접근 권한은 이용자가 영상 촬영 기능을 사용할 때에만 요청됩니다.
- 촬영된 영상은 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.
- 카메라/마이크 접근 권한은 기기의 설정 메뉴에서 언제든지 허용 또는 해제할 수 있습니다.
- 접근 권한을 해제하더라도 영상 촬영 외의 다른 서비스 기능(기록 조회, 캘린더 등)은 정상적으로 이용할 수 있습니다.

## 9. 개인정보의 안전성 확보 조치

앱은 이용자의 개인정보를 안전하게 관리하기 위하여 다음과 같은 조치를 취하고 있습니다.

### 9-1. 기술적 조치

- **데이터 암호화:** 모든 데이터는 전송 시 TLS(HTTPS)를 통해 암호화되어 전송됩니다.
- **접근 제어:** Supabase의 Row Level Security(RLS) 정책을 통해 이용자 본인의 데이터에만 접근할 수 있도록 제한합니다.
- **인증 토큰 관리:** JWT(JSON Web Token) 기반의 인증 체계를 사용하며, 토큰은 안전하게 관리됩니다.
- **영상 파일 로컬 저장:** 클라이밍 영상은 사용자 기기에만 저장되며, 외부 서버로 전송되지 않아 유출 위험이 없습니다.

### 9-2. 관리적 조치

- 개인정보에 대한 접근 권한을 최소한의 인원으로 제한합니다.
- 개인정보 보호에 관한 법률 및 내부 방침을 정기적으로 점검합니다.
